# DumpIt Frontend

React single-page application for the DumpIt anonymous text-sharing platform.

## Tech Stack

- **Framework**: React 18 + Vite
- **Routing**: React Router v6
- **State**: React Context (AppStore)
- **Styling**: Custom CSS with CSS variables

## Features

- Anonymous session management
- Public feed of visible dumps
- Create, edit, delete dumps and comments
- AI moderation status tracking
- Mobile responsive sidebar with hamburger menu
- Error boundary for crash recovery
- Configurable API endpoint

## Development

```bash
npm install
npm run dev
```

App runs at `http://localhost:5500`.

## Build

```bash
npm run build
npm run preview
```
